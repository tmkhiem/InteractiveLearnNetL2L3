# InteractiveLearnNetL2L3

A static HTML/CSS/JavaScript learning tool for visualizing:

- Layer 2 MAC addressing and switching
- Layer 3 IP addressing and routing
- Encapsulation and decapsulation
- ARP lookup and ARP learning
- Routing table lookup

## Run locally

Open `/home/runner/work/InteractiveLearnNetL2L3/InteractiveLearnNetL2L3/index.html` in a browser, or serve the folder as static content.

## Controls

- Start
- Pause
- Resume
- Step forward
- Reset
- Speed: Slow / Normal / Fast
- Previous lesson / Next lesson
